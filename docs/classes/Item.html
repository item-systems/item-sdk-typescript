Class Item

The ITEM class is the primary interface point for the digital twin of an NFI. Use this class to execute standard non-fungible token interations as well as additional capabilities like authentication and configuration. WalletConnect 2.0 has native support through the neon-invoker package.

To use this class:

import { Item } from '@item-systems/item'
import Neon from '@cityofzion/neon-core'
import { NeonInvoker } from '@cityofzion/neon-invoker'
import { NeonParser } from '@cityofzion/neon-parser'

const account = new Neon.wallet.Account()

const item = new Item({
  scriptHash,
  invoker: await NeonInvoker.init(NODE, account),
  parser: NeonParser,
})
const totalItems = await item.totalSupply()
console.log(totalItems)
