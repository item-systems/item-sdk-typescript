    createEdge

    • createEdge(
          params: {
              description: string;
              entryPoint: number;
              exitPoint: number;
              questId: number;
          },
      ): Promise<string>

      Creates a new edge entity in the system. Initially edges are set to restricted (only the creator can participate). Edges can only be attached to nodes which are either public or owned by the creator. Additionally, they can only be added to quests which share a creator. Edges are vectored and require an entry node and exit node.

      Parameters

      • params: { description: string; entryPoint: number; exitPoint: number; questId: number }
        • description: string

          A description of the edge

        • entryPoint: number

          the node that the edge can be entered from

        • exitPoint: number

          the node that the edge exits to

        • questId: number

          The quest to attach the edge to

      Returns Promise<string>
